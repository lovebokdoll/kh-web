
 변수
 :특정 데이터 값을 임시로 저장하는 공간

 이유
 :반복사용(일괄처리),자주사용
 :반복적인 구문에서 여러 코드에 걸쳐서 사용될 때 

 주의사항 
 :1.숫자로 시작 불가
 :2.특수문자로 삽입불가
 3.예약어 사용불가
 4.대소문자 구분

 변수의 유효범위
 1.전역변수 -window
 :코드블록 밖에서 선언된 변수
 : 해당코드 전역에서 접근가능
 : 여러 개의 서로 다른 함수들이 같은 값을 공유해야 할 떄
:전역변수 많아지면 전역변수 오염발생

2.지역변수-코드블록 안에서만 사용
:코드 블록 안에서 선언된 변수
:해당 코드블록 안에서만 접근가능
:변수 값이 해당 코드 블록한에서만 사용가능
